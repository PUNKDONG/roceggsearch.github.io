# 洛克王国查蛋助手

这是一个适合直接发布到 `GitHub Pages` 的纯前端版本。

## 文件

- `index.html`
- `styles.css`
- `app.js`
- `data/pets.csv`
- `.nojekyll`

## 部署到 GitHub Pages

1. 新建 GitHub 仓库
2. 把本目录所有文件上传到仓库根目录
3. 打开仓库 `Settings -> Pages`
4. 选择：
   - `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. 保存后等待发布

发布地址通常是：

```text
https://你的用户名.github.io/你的仓库名/
```
